# Java-Chat-System

This is a minimal Java Swing application packaged with Maven.

Build and run (Windows cmd.exe):

```
cd /d C:\Users\quoca\source\Java\Java-Chat-System
javac -d out src\\*.java
mvn -B package
java -cp target\classes App
```

Notes:
- Requires JDK 11 or newer and Maven installed.
- The application is a simple Swing frame to demonstrate the project structure.
